Requirements for running the code contained in this project

Python 3.9.19  
Packages:  
- matplotlib==3.9.2  
- seaborn==0.13.2  
- numpy==1.26.4  
- pandas==2.2.2  
- scikit-learn==1.5.1  
- xgboost==2.1.3  

You can install them using the following pip command:  

pip install matplotlib==3.9.2 seaborn==0.13.2 numpy==1.26.4 pandas==2.2.2 scikit-learn==1.5.1 xgboost==2.1.3
